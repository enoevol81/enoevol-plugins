# Reference Image Index

Curated visual exemplars of Swiss / International Typographic Style, one (or a few) per principle. Filenames are prefixed with the principle number they best demonstrate, matching the pairing in the source document. View these to ground compositional choices in canonical work.

## Overview

- `00-overview-grid-dark.jpeg` — “Swiss Style” poster with visible column/baseline grid overlay. The grid as the design.
- `00-overview-appropriation-restraint.webp` — AGDA “Appropriation And Restraint” lecture poster + grid breakdown (Mark Lobo, 2017). Modular grid, color block as structure.
- `00-overview-geometric-color.jpeg` — Geometric “Facere Possimus” poster pair. Reductive shape + limited palette.
