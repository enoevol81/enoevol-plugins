# Annotated Canon — Swiss Reference Works

Curated exemplars of Swiss / International Typographic Style, one (or a few) per principle. Filenames are prefixed with the number of the principle they best demonstrate. **The image files are real and sit in this directory — open them with the Read tool to look at them** when calibrating a composition. Each entry below also describes, in text, exactly what to imitate — grid structure, scale contrast, spatial division — so the lesson survives even without viewing.

## Overview works

### `00-overview-grid-dark.jpeg` — "Swiss Style" poster with exposed grid
Column/baseline grid drawn visibly over the finished poster. **Imitate:** the grid as the first artifact, not a rationalization — every element's edges land on the drawn lines, and empty modules are left deliberately empty. Use this to show a client (or yourself) what "the grid is the design" means.

### `00-overview-appropriation-restraint.webp` — AGDA "Appropriation And Restraint" lecture poster + grid breakdown (Mark Lobo, 2017)
Modern proof the system still works. **Imitate:** a solid color block used as *structure* (it occupies whole grid modules and creates the composition's mass), not decoration; a modular grid where text blocks and the color field share the same cells; hierarchy carried by one big scale jump.

### `00-overview-geometric-color.jpeg` — geometric "Facere Possimus" poster pair
Reductive shape + limited palette. **Imitate:** how few moves a poster needs — one geometric idea, two or three flat colors, small flush-left type block anchoring a corner. The pair also shows consistency: same system, two outputs.

## Per principle

### 1 · Grid Systems — `01-grid-layout-levels.webp`
Layout-levels diagram: the *same* grid holding an escalating number of images (calm → balance → power). **Imitate:** density as a variable *within* one grid — you change the energy of a layout by how many modules you fill, never by changing the structure. This is the template for building a series (campaign, deck, feed) on one system.

### 2 · Alignment — `02-alignment-vision.jpeg`
"vision" poster — repeating diamond module. **Imitate:** every element hangs on a shared axis; the module repeats at one size and one rhythm, so a single alignment decision governs the whole field. Count the distinct alignment axes: there are almost none, which is the point.

### 3 · Hierarchy — `03-hierarchy-helvetica.jpeg`
Helvetica / Max Miedinger / 1957 study poster. **Imitate:** read order manufactured with only scale + weight + position — the typeface name enormous, the metadata small and flush to a column edge. Note the size ratio between the dominant word and the supporting line: multiples, not increments.

### 4 · Typography First — `04-typography-beethoven.jpeg`
"beethoven" Tonhalle concert poster (Josef Müller-Brockmann, 1955). **Imitate:** the whole message carried by one word of type plus geometry — the arcs are structure (rhythm made visible), not illustration. Cover the arcs and the poster still works; that is the "would it survive without imagery" test passing. Also study the tiny, dense information block: all logistics in one small flush-left cluster, refusing to compete with the dominant element.

### 5 · Asymmetrical Balance — `05-asymmetry-die-zeitung.jpeg`
"die Zeitung" exhibition poster (Gewerbemuseum Basel, 1958). **Imitate:** one giant letterform + photo massed to one side, countered not by another object but by whitespace with a small text anchor. Visual weight ≈ size × darkness: the small dense block balances the huge form because the void amplifies it.

### 6 · Contrast — `06-contrast-grid-templates.jpeg`
A4 grid templates on black. **Imitate:** figure/ground contrast doing hierarchy's work — the light grid diagrams pop from the black field with no added decoration; and column-density vs. baseline-density as two contrast axes you can vary independently.

### 7 · White Space — `07-whitespace-velvet.jpeg`
Velvet Underground / EPI-style poster. **Imitate:** geometric forms given room to breathe — the negative space is the staging that makes the small forms read as deliberate. Notice the between-element gaps are visibly larger than any within-element gap; proximity, not boxes, does the grouping.

### 8 · Consistency — `08-consistency-wireframes.webp`
Hand-drawn editorial layout thumbnails. **Imitate:** one system applied across many spreads — same columns, same slots for image/heading/body, varied only in which modules each spread fills. This is what to produce *before* designing any multi-page or multi-screen artifact: thumbnails proving the system extends.

### 9 · Restraint — `09-restraint-swiss-white.jpeg`
"Swiss Style" poster pair on white. **Imitate:** maximum reduction — type and space only, one weight jump, one alignment axis, enormous margins. The benchmark to hold a draft against when deciding whether the last three elements earn their place.

## Notes

- Attributions are best-effort from the source material; treat the *principle* each work teaches as the reliable part, not the provenance.
- These are study references, not assets to reproduce in deliverables. Use them to calibrate the eye, then build original work.
